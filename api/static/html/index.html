Routes Description!

User Routes

Method Path Description
GET /users fetch all the users
GET /user/:id fetch a single user
POST /register create a new user
POST /login allow user to login to page using email and password
PUT /user/:id update a user's details
PATCH /user/:id modify a user's details
DELETE /user/:id delete a user

Product Routes

Method Path Description
GET /products fetch all products
GET /product/:id fetch a single product
POST /product add a new product
PUT /product/:id update a product's details
PATCH /product/:id modify a product's details
DELETE /product/:id delete a product
